cordovaNetwork

Fetches the current network available (WIFI, 3G, etc.). The directive keeps updating the scope on network changes.

Installation:

cordova plugin add org.apache.cordova.network-information

Tested against plugin version r0.2.15 specification.

Official Docs:

https://github.com/apache/cordova-plugin-network-information/blob/master/doc/index.md

Usage:

<ANY cordova-network>
...
</ANY>

Examples:

<div cordova-network>
Current network: {{network}}
</div>
