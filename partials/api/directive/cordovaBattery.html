cordovaBattery

Fetches the current battery status. The directive keeps updating the scope on battery changes.

Installation:

cordova plugin add org.apache.cordova.battery-status

Tested against plugin version r0.2.12 specification.

Official Docs:

https://github.com/apache/cordova-plugin-battery-status/blob/master/doc/index.md

Usage:

<ANY cordova-battery>
...
</ANY>

Examples:

<div cordova-battery>
Battery level: {{battery.level}}
Is device charging?: {{battery.isPlugged}}
</div>
