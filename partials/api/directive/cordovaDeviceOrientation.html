cordovaDeviceOrientation

Checks direction or heading that the device is pointed. The directive watches the direction and update the scopes on changes.

Installation:

cordova plugin add org.apache.cordova.device-orientation

Tested against plugin version r0.3.11 specification.

Official Docs:

https://github.com/apache/cordova-plugin-device-orientation/blob/master/doc/index.md

Usage:

<ANY cordova-device-orientation="expression">
...
</ANY>

Params:

Param Type Details
cordovaDeviceOrientation expression

Expression which evaluates to an object that represent update options with 'filter' or 'frequency' as keys. Check the official guide for more info. Default value is frequency=100. see examples.

Examples:

<div cordova-device-orientation>
Current heading is updating every 100 millis and is {{deviceOrientation.magneticHeading}}.
</div>

<div cordova-device-orientation="{frequency: 500}">
Current heading is updating every 500 millis and is {{deviceOrientation.magneticHeading}}.
</div>

<div cordova-device-orientation="{filter: 30}">
Current heading is {{deviceOrientation.magneticHeading}}.
Only updates when the change in degrees is over 30.
</div>
