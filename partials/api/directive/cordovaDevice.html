cordovaDevice

Fetches the device information.

Installation:

cordova plugin add org.apache.cordova.device

Tested against plugin version r0.3.0 specification.

Official Docs:

https://github.com/apache/cordova-plugin-device/blob/master/doc/index.md

Usage:

<ANY cordova-device>
...
</ANY>

Examples:

<div cordova-device>
Device model: {{device.model}}
</div>
