cordovaAppAvailability

Checks installed applications

Installation:

cordova plugin add https://github.com/ohh2ahh/AppAvailability.git

Tested against plugin version 0.3.1 specification.

Official Docs:

https://github.com/ohh2ahh/AppAvailability

Usage:

<ANY cordova-app-availability="expression">
...
</ANY>

Params:

Param Type Details
cordovaAppAvailability expression

Expression which evaluates to an object, an array or a string. When string, it is being checked as uri scheme and result accessible from scope.appAvailability. When object, it's values are being checked and accessible as object whose keys are the corresponding keys from the original object. When array, it's elements are being checked and accessible as array whose elements are the results in the same order as the original array. see examples.

Examples:

<div cordova-app-availability="'twitter://'">
Twitter is {{appAvailability ? '' : 'not'}} available.
</div>

<div cordova-app-availability="{twitter: 'twitter://', facebook: 'fb://'}">
Twitter is {{appAvailability.twitter ? '' : 'not'}} available.
Facebook is {{appAvailability.facebook ? '' : 'not'}} available.
</div>

<div cordova-app-availability="['twitter://', 'fb://']">
Twitter is {{appAvailability[0] ? '' : 'not'}} available.
Facebook is {{appAvailability[1] ? '' : 'not'}} available.
</div>
